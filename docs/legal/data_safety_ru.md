# Data Safety Disclosure (Google Play) — TOCHKA

Последнее обновление: 15 апреля 2026

## 1) Категории данных, которые собираются

- Контактные данные: email.
- Пользовательский контент: данные запроса провайдеру (если пользователь отправляет запрос).
- Данные устройства: модель устройства, версия ОС, технические идентификаторы приложения/сессии.
- Диагностика: crash-логи, технические журналы ошибок.

## 2) Для чего используются данные

- Функциональность приложения (каталог, фильтры, аккаунт).
- Передача запроса выбранному пользователем провайдеру.
- Поддержка пользователей.
- Безопасность и предотвращение злоупотреблений.
- Аналитика и улучшение качества приложения (в агрегированном виде).

## 3) Передача данных

Данные могут передаваться:
- выбранному пользователем провайдеру (только в объеме запроса);
- техническим подрядчикам (хостинг, мониторинг, аналитика) по договору.

## 4) Привязка данных к пользователю (Linked / Not linked)

- Linked (привязанные): email, данные запроса, данные аккаунта.
- Not linked (непривязанные/агрегированные): часть технической аналитики и диагностики, если применяется обезличивание.

## 5) Чего TOCHKA не делает

- не продает персональные данные;
- не использует данные для стороннего рекламного трекинга.

## 6) Удаление данных

Пользователь может инициировать удаление аккаунта и связанных данных:
- в приложении: Settings -> Account -> Delete account;
- через web: https://thetochka.com/delete-account;
- по email: delete@thetochka.com.
