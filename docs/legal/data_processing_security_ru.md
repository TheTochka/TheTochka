# Data Processing & Security — TOCHKA (RU)

Последнее обновление: 15 апреля 2026

## 1. Технические меры защиты

TOCHKA применяет:
- HTTPS/TLS для передачи данных;
- контроль доступа по ролям и принцип минимально необходимого доступа;
- журналирование технических событий безопасности;
- мониторинг и ограничение аномальной активности.

## 2. Организационные меры

- доступ к данным предоставляется только уполномоченным сотрудникам/подрядчикам;
- подрядчики обязаны соблюдать договорные требования по защите данных;
- периодически пересматриваются сроки хранения и объем обрабатываемых данных.

## 3. Хранение

Данные хранятся только в пределах сроков и целей, необходимых для работы сервиса и исполнения требований закона.

## 4. Передача третьим сторонам

Передача данных третьим сторонам ограничивается:
- выбранным пользователем провайдером (для обработки запроса);
- техническими сервисами (хостинг, мониторинг, аналитика), необходимыми для работы платформы.

## 5. Инциденты

При выявлении инцидентов безопасности TOCHKA предпринимает меры по локализации, анализу и снижению возможного ущерба в разумные сроки.
