# Политика удаления данных TOCHKA (RU)

Последнее обновление: 15 апреля 2026

## 1. Как подать запрос

Пользователь может запросить удаление аккаунта и связанных данных:
- через интерфейс приложения (Settings -> Account -> Delete account);
- через web: https://thetochka.com/delete-account;
- по email: delete@thetochka.com.

## 2. Что удаляется

По запросу удаляются данные, связанные с аккаунтом пользователя, включая:
- профильные данные аккаунта;
- технические привязки аккаунта;
- историю обращений, если ее хранение не требуется законом.

## 3. Что может храниться после удаления

Мы можем сохранить ограниченный объем данных, если это необходимо:
- для выполнения требований закона;
- для предотвращения мошенничества и инцидентов безопасности;
- для защиты прав и законных интересов TOCHKA.

## 4. Срок обработки

Стандартный срок обработки запроса — до 30 календарных дней после подтверждения запроса и верификации пользователя.

## 5. Контакты

Канал для удаления данных: delete@thetochka.com
