## О сервисе

**TheTochka** — мобильный **маркетплейс VPN-провайдеров** для iOS и Android: каталог с фильтрами, независимые отзывы, сравнение условий и запрос КП прямо из приложения. Это **нативное приложение**, а не сайт-агрегатор.

**TheTochka не продаёт VPN-подключение напрямую** — платформа даёт информацию и инструменты сравнения; договор и оплата — между вами и выбранным провайдером.
